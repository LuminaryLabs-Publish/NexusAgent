# Public Roadmap

## Phase 1: Public bootstrap

- Publish clean repository structure.
- Add public-safe docs, examples, schemas, and harness placeholders.
- Keep private Dev state out of the public repo.

## Phase 2: Simple harness release

- Add reviewed runnable Simple harness source.
- Add minimal tests.
- Add one local-model smoke example.

## Phase 3: Lite harness release

- Add reviewed runnable Lite harness source.
- Add patch/write/validate examples.
- Add public-safe action schema coverage.

## Phase 4: Orchestration docs

- Publish sanitized Orchestrator architecture docs.
- Add example packets and ledgers.
- Keep internal benchmark state private.

## Phase 5: Public CI smoke

- Add a short local-model-compatible smoke workflow.
- Keep model downloads optional and public-safe.
- Avoid committing generated artifacts.
