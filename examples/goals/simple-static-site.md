# Example Goal: Simple Static Site

Build a small static website inside the allowed workspace.

## Requirements

- Create `index.html`.
- Create `styles.css`.
- Create `main.js` only if needed.
- Include a visible title, short description, and one interactive element.
- Keep all files inside the workspace.
- Do not use external services.
- Add a short validation note explaining how to open the page locally.

## Success criteria

- Files are present.
- HTML is readable.
- No secrets or external endpoints are introduced.
- The output can be inspected in a browser.
